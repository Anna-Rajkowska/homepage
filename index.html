HTML składa się ze znaczników/tagów/elementów

budowa znaczników:

znacznik atrybut="wartość">treść / znacznik>

To co widzimy w przeglądarce jest oznaczone jako treść, znaczniki i tagi są czytane i analizowane przez przeglądarkę i nie są one widoczne dla użytkownika

Semantyczny - mający znaczenie

Pliki HTML mają rozszerzenie .html

HTML to szkielet strony, jak u ludzika :D składa się z: head, body,

znaczniki html

lang= - język

main> - główna część dokumentu

section> - sekcje dokumentu

article> - taka sekcja, która może istnieć samodzielnie

Sekcja to jak np. sekcja sport w gazecie, artykuł to samodzielna grupa treści i może występować samodzielnie. Każda sekcja powinna mieć nagłówek

header> - naglówek dokumentu

footer> stopka sekcji lub artykułu

h1-h6> - nagłówki

p> - paragraf i akapit

link - dołącza link do strony

Tworzenie nowej strony:

1. nowy folder z nazwą, np. HOMEPAGE

2. nowy plik w środku - index.html

3. Żeby zacząć tworzyć nową stronę należy w Visual Studio COde wcisnąć ! + tab i program sam tworzy szkielet strony.

4. Pamiętać aby zmienić język strony z angielskiego na używany

5. Dodajemy treść, dzielimy ją na nagłówki, artykuły, paragrafy itp.

6. Porządkujemy kod wciskając ctrl + shift + p i wybieramy format document, a VSC robi całą magię za nas ;)
